# Custom APIs for Joomla – REST API Builder & Endpoint Manager
Joomla Utilities: Custom API Extension

Streamline data management with **Custom APIs for Joomla**, a powerful plugin that enables you to create and manage custom endpoints and REST APIs directly on your Joomla website. With an intuitive GUI, you can seamlessly interact with Joomla database tables—fetching data such as user roles, groups, custom fields, or featured images—and perform **Create, Read, Update, Delete (CRUD)** operations effortlessly.

## Key Highlights:
- **Custom API Creation**:  
  Build and manage REST API endpoints without writing complex SQL queries.
- **Data Flexibility**:  
  Fetch diverse data types including user profiles, articles, groups, or custom fields using simple filters or custom SQL.
- **External API Integration**:  
  Connect third-party APIs and display external platform data on your site via customizable shortcodes.
- **Secure Access**:  
  Enhance API security with **Bearer Token Authentication** for protected data exchange.

## Quick Setup:
1. Go to **Extensions → Manage → Install**  
2. Search for **"Custom APIs for Joomla"** and install the plugin  
3. Configure and customize your APIs using the plugin’s GUI
